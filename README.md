# Campaign statistics analysis for Vinix7

## 📌 Project Overview
This project analyzes the effectiveness of two marketing promotion strategies: 20% Discount and Cashback 50K. The goal is to determine which strategy generates higher revenue and to evaluate the impact of advertising spend on website traffic.

## 🎯 Objectives
- Compare the performance of two promotional strategies using statistical testing
-Identify whether the revenue difference is statistically significant
- Analyze the relationship between advertising spend and web traffic

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy, SciPy, Matplotlib/Seaborn)
- Google Colab
- Statistical Methods (Hypothesis Testing, Linear Regression)

## 📊 Key Insights
- The 20% discount strategy significantly outperforms cashback in generating revenue
- Statistical testing confirms the difference is not due to chance (p-value < 0.05)
- Advertising spend has a positive correlation with web traffic
